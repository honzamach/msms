Roles
================================================================================

According to the Ansible `role documentation <https://docs.ansible.com/ansible/latest/user_guide/playbooks_reuse_roles.html>`_
and `best practices <https://docs.ansible.com/ansible/latest/user_guide/playbooks_best_practices.html>`_,
playbooks implementing particular feature should be organized into roles. In that
way they are easilly reusable for other users and all corresponding tasks, variables
and metadata are neatly organized within single directory structure.
